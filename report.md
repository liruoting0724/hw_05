# 实验报告：极简CNN与LeNet-5手写数字识别

## 一、文章学习摘要（任务一）
本文实现极简CNN用于MNIST手写数字识别。
模型结构：输入28×28灰度图 → 卷积层 → ReLU → 最大池化 → 展平 → 全连接层输出10分类。
训练流程：加载MNIST数据集 → 数据归一化 → 定义模型、损失函数、优化器 → 训练5轮 → 测试评估 → 保存模型。

## 二、LeNet-5结构说明（任务二）
| 层类型 | 输入尺寸 | 输出尺寸 | 说明 |
|--------|----------|----------|------|
| 输入层 | 1×28×28 | 1×28×28 | MNIST灰度图 |
| 卷积层1 | 1×28×28 | 6×28×28 | 5×5卷积核，padding=2 |
| 池化层1 | 6×28×28 | 6×14×14 | 2×2平均池化 |
| 卷积层2 | 6×14×14 | 16×10×10 | 5×5卷积核 |
| 池化层2 | 16×10×10 | 16×5×5 | 2×2平均池化 |
| 展平层 | 16×5×5 | 400维 | 转为一维向量 |
| 全连接层1 | 400 | 120 | 全连接 |
| 全连接层2 | 120 | 84 | 全连接 |
| 全连接层3 | 84 | 10 | 输出10分类 |

## 三、实验结果
- 极简CNN测试准确率：约98.1%
- LeNet-5测试准确率：约98.5%

## 四、对比分析
LeNet-5拥有更深的卷积层与全连接层，特征提取能力更强，准确率略高；
极简CNN结构更轻量，训练更快，参数量更少，适合入门理解CNN原理。
